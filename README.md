Navigate to https://pcbtal.github.io/time-qr-VM-1.0/ to access a QR code that updates every second with the following output timedatectl set-time "YYYY-MM-DD HH:MM:SS"
  Where YYYY-MM-DD HH:MM:SS is the current system time and date.
Updates every 3 seconds for compatibility issues with dim screens and various scanners.
